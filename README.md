# Installation :

    $ git clone https://github.com/ThomasDruart/Checkpoint4-Lebonplaneur.git
    $ installer la BDD (sql)
    $ cd Checkpoint4-Lebonplaneur/backend
    $ npm install
    $ npm start
    $ cd Checkpoint4-Lebonplaneur/frontend
    $ npm install
    $ npm start
